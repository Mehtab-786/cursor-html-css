# Cursor UI

A modern landing page for the Cursor product, built with HTML and CSS.

## Getting Started

1. Clone or download this repository.
2. Open `index.html` in your browser to view the UI.

## Project Structure

- `index.html` — Main HTML file
- `style.css` — Custom styles
- `assets/` — Images and SVGs
- `fonts/` — Custom font files

## Customization

- Update images in the `assets` folder as needed.
- Modify colors and fonts in `style.css` for your branding.
